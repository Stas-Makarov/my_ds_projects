# Исследовательский анализ набора фотографий.

Подготовить модель, предсказывающую возраст по фотографиям, метрика MAE не должна превышать 8.

## Общий вывод:

Удалось достигнуть поставленной задачи (метрика МАЕ менее 8) в результате следующих действий:

- Применили архитектуру ResNet50 (задали функцию активации ReLU, функцию потерь - среднеквадратичную ошибку, один выходной нейрон, поскольку задача регрессии)
- Перед обучением модели к обучающей выборке применили аугментацию (поворот изображений, отражения изображений относительно осей, сдвиги изображений), что также положительно сказалось на качестве предсказаний, т.к значительно увеличило обучающую выборку.
- Методом ручного подбора задали Learning rate=0.00005 и количество эпох, равное 6.
